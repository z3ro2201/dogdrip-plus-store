차단 키워드

🚫 조건별 키워드 차단 목록 관리

등록된 키워드별로 **타겟 영역(전체/댓글/게시글)** 및 **매칭 방식(포함/시작단어)** 조건에 맞춰 목록을 정밀 필터링합니다.

👤 차단 된 사용자 목록 관리

이곳에 설정 된 사용자는 게시글 및 댓글 목록에 보이지 않게 됩니다.

📝 유저 메모 목록 관리

개드립에서 등록했던 사용자들의 고유 메모 리스트입니다.
등록된 메모 배지를 누르면 내용을 간편하게 즉시 수정하거나 삭제할 수 있습니다.

🖼️ 차단된 개드립콘 (단일 파일 기준)

차단된 이모티콘은 본문이나 댓글창에 보이지 않습니다.

📦 차단된 개드립콘 (패키지 그룹 기준)

해당 개드립콘 전체가 본문이나 댓글창에 보이지 않습니다.

📐 레이아웃 숨김 및 특수 필터 제어

체크된 항목의 경우 게시물 목록이나 메인에서 보이지 않게 됩니다.

🚫 사용자 차단 처리 방식

💾 데이터 백업하기

개드립콘 절약모드 혹은 색상반전 모드를 제외한 차단 대상과 기본 설정값이 저장됩니다.

💾 데이터 복원하기

기존에 다운로드 받아둔 백업(.json) 파일을 불러와 시스템의 차단 구성과 레이아웃 비율을 원상복구합니다.

🔗 Dogdrip++ 데이터 가져오기

타 확장프로그램(Dogdrip++)의 데이터 중 차단 사용자 및 키워드 데이터만 복원합니다.

💡 개드립 Plus + 에 대하여

개드립 Plus + 는 더 많은 분들이 개드립넷을 편리하고 쾌적하게 이용할 수 있도록 만들어진 확장프로그램입니다.

기존에 파편화되어 있던 여러 편의 기능들을 하나의 패키지로 통합했습니다. 유저·키워드 차단부터 레이아웃 커스터마이징까지, 별도의 확장프로그램을 여러 개 설치할 필요 없이 이 하나로 해결할 수 있도록 하는 것이 목표입니다.

유용하게 사용해 주셔서 감사합니다.
